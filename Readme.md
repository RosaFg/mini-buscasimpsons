# Mini Simpsons Search

Es un pequeño proyecto que hice durante el testeo de buscasimpsons.com

aprendizaje sobre consumo de APIs REST. Usa la API pública de [buscasimpsons.com](https://buscasimpsons.com) para buscar citas y fotogramas de la serie.

## Que hace

- Busca frases, Escenas o episodios en la API
- Muestra los resultados en tarjetas con imagen, episodio, cita y minuto
- Carga los resultados de a 8 con un boton "Cargar mas"

## Tecnologias

- HTML, CSS y JavaScript vanilla
- Fetch API para las peticiones HTTP
- Sin dependencias ni frameworks

## Como usarlo

Abre el archivo `index.html` directamente en el navegador. No requiere servidor ni instalacion.

## API

Endpoint utilizado: `https://api.buscasimpsons.com/search/{termino}`

Documentacion: [buscasimpsons.com](https://buscasimpsons.com)
